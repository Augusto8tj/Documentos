# **App Name**: Document Flow

## Core Features:

- Automatic Numbering: Automatic numbering of documents (Ofícios, Memorandos, Portarias, Decretos, Atas, etc.).
- Document Interface: User interface for document creation, editing, and sharing.
- Google Docs Integration: Backend integration with Google Docs for document storage and versioning.
- Sharing: Sharing options for documents, with permission controls

## Style Guidelines:

- Primary color: Soft blue (#7CB9E8) to convey a sense of calmness, focus and reliability.
- Background color: Light gray (#F0F0F0) to provide a clean and neutral backdrop.
- Accent color: Muted green (#A3E4D7) to highlight key actions.
- Use clear and professional fonts such as Roboto or Open Sans for readability.
- Employ simple and intuitive icons to represent document actions (e.g., edit, share, download).
- Maintain a clean and organized layout with clear hierarchy and spacing for ease of navigation.
